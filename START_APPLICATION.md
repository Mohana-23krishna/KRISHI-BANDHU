## Start the Application

### Windows (PowerShell)
```powershell
cd krishibandhu
mvn clean install
mvn spring-boot:run
```

### Linux/Mac
```bash
cd krishibandhu
mvn clean install
mvn spring-boot:run
```

**Wait 30-60 seconds** for the application to start.

## Access Points

Once started, access:

1. **Homepage**: http://localhost:8080
2. **Voice Test Page**: http://localhost:8080/voice-test
3. **Buyer Dashboard**: http://localhost:8080/buyer
4. **Farmer Registration**: http://localhost:8080/farmer-register

## Quick Test

Open browser and go to: http://localhost:8080/voice-test

Try saying or typing:
- "I have 20 kgs of tomatoes from Hyderabad"
- Click "Test Voice API"
- Check the response and view created crops!
